🚀 AI Bug Fixer Pro
Ek full-stack web application jo developers ko unke code mein bugs pehchanne aur unhe fix karne mein madad karta hai. Isme modern AI ka use kiya gaya hai taaki code logic ko asani se samjha ja sake.

✨ Key Features
Secure Login: Firebase Authentication ka use karke Email/Password aur Google/GitHub login support.

AI Debugging: GPT-4o-mini model ka use karke bugs ko instantly fix karna.

Code Explanation: Sirf solution hi nahi, balki AI batata hai ki galti kyun thi aur logic kya hai.

Dark & Light Mode: User ki preference ke hisab se sleek UI themes.

Fully Responsive: Mobile aur Desktop dono par perfect chalta hai.

🛠 Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+).

Backend: Vercel Serverless Functions (Node.js runtime).

Auth & BaaS: Firebase.

AI Engine: OpenRouter API (GPT-4o-mini).

🛡 Security First
Is project mein Security ka khaas dhyan rakha gaya hai:

API Keys ko frontend par expose nahi kiya gaya hai.

Ek Backend Proxy (Serverless Function) ka use kiya gaya hai jo API calls ko secure handle karta hai.

Sensitive data ko Environment Variables mein rakha gaya hai.

🚀 How to Run Locally
Clone the repo:

Bash
git clone https://github.com/YourUsername/ai-bug-fixer-pro.git
index.html ko browser mein open karein (lekin AI calls ke liye aapko Vercel par deploy karna hoga).

Vercel par deploy karte waqt OPENROUTER_API_KEY ko environment variables mein add karein.
